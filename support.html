Trip Dashboard — Support

A private, offline-first travel planner: organise places, build day-by-day plans, share to Google Maps, generate AI itineraries, and split trip expenses.

Quick help

  • Import places: Places tab → Import → pick an Excel/CSV/JSON file.
  • Add from Google Maps: in Google Maps, Share → Trip Dashboard, then open the app.
  • Build a day plan: Trips tab → open a trip → add places to days.
  • AI trips: AI tab → add your provider's API key → type cities → Generate.
  • Split expenses: Expenses tab → new group → add members and expenses.
  • Back up: Places tab → “…” → Back up (saves favourites, trips, expenses, photos).
